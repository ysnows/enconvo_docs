Enconvo 2.2.1 Changelog (2025-06-07) 🚀


⚠️ Important Warning

Due to the need to enable the new Credentials management system, previously configured settings need to be reconfigured, and previously created Commands will be lost. Please save your currently used Prompts and API keys before upgrading.


🔄 Workflow - Build your AI automation workflow using built-in tools and numerous MCP Servers

🎨 Visual Workflow Editor

Brand new visual workflow editor, clearer, more intuitive and faster workflow construction

🛠️ Built-in Tools & MCP Servers

Built-in 200+ tools, native support for using MCP Servers in workflows, making workflow extensibility endless

🧠 Smart Parameters Support

Can automatically generate tool parameters based on workflow context through AI

📦 Workflow Templates Store

New Workflow Templates Store, users can quickly install templates with built-in default templates

⚙️ Jinja2 Template Support

Workflow parameter settings support jinja2 template syntax, making workflow parameter configuration more flexible


🔐 Brand new Credentials Management

🔧 Optimized Management Logic

Optimized Credentials management logic, making Credentials management more convenient

🔑 OAuth 2.0 Support

Support for OAuth 2.0 authentication

📋 More Provider Credentials

Added more Provider Credentials management


🔌 New Plugins

📧 Gmail

New Gmail extension, supports OAuth2 authorization login, supports sending emails, reading emails and other tools

🕷️ Firecrawl

New Firecrawl extension (including Enconvo Cloud support), supports web scraping, website crawling and other tools

🎙️ Gemini TTS

New Gemini TTS, Gemini TTS Multi Speaker extensions (including Enconvo Cloud support), supports multi-person conversation TTS generation
